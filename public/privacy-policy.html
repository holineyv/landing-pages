Privacy Policy for RaptorLink

This privacy policy explains how the RaptorLink browser extension ("we", "our", or "the Service") collects and uses information when you use our Service.

Information Collection and Use

The RaptorLink extension collects and uses certain information to provide its link building and content analysis features. Here's what we collect and how we use it:

1. Page Content and HTML

  • We analyze the HTML content of pages you visit when using the extension's features
  • This includes text content, meta information, and link structures
  • This data is processed locally in your browser and is not sent to our servers
  • We use this to:
    • Analyze keyword relevance
    • Find suitable link placement opportunities
    • Detect external links
    • Calculate content metrics

2. URLs and Domain Information

  • We collect URLs of pages you analyze
  • This includes:
    • Pages where you search for link placements
    • Sitemap URLs you provide
    • Landing pages you save
  • This data is stored in your browser's local storage
  • We use this to:
    • Find matching content
    • Suggest link placements
    • Save your insertion history

3. OpenAI API Integration

  • When you use the AI rewrite feature, your selected text is sent to OpenAI's API
  • This is only done when you explicitly request a rewrite
  • We use your provided OpenAI API key for these requests
  • The data sent includes:
    • The original sentence you want to rewrite
    • The anchor text you want to integrate

4. Local Storage

  • We store certain data locally in your browser, including:
    • Your saved URLs and landing pages
    • Link insertion history
    • Extension settings
    • Parsed sitemap results
  • This data remains on your device

Data Security

  • Local data remains in your browser until you clear it
  • When you use the link sharing feature, only essential link placement details are transmitted securely to our servers
  • We do not collect or store any personal browsing history or analysis data

Third-Party Services

The extension integrates with:

  • OpenAI API (for text rewriting)
  • You control access to this service through your own API key

Changes to This Privacy Policy

We may update our Privacy Policy from time to time. We will notify you of any changes by updating the policy in the extension. Continued use of the Service after changes means you accept the updated policy.
